Instruções para uso do jogo de bisca desenvolvido em C. Compatível com Windows e Linux.
Pode ser jogado contra um computador ou em duplas (jogador + computador vs. 2 computadores).

Primeiramente, baixe o repositório como .zip e extraia-o, ou clone o repositório usando o comando git clone. Depois, abra o terminal do seu sistema operacional e navegue até a pasta do repositório utilizando o comando cd com o endereço da pasta entre aspas. Rode este comando no terminal para certificar de que o jogo está compilado corretamente:

make

Depois, basta utilizar o comando make run no mesmo diretório ou executar o arquivo "bisca" (para Windows) para jogar.
